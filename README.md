WUMPUS WORLD KNOWLEDGE BASED AGENT

PROJECT OVERVIEW
This is a web based simulation of the classic Wumpus World AI problem. It features an automated agent that uses propositional logic and CNF resolution to safely navigate a grid, avoid pits and the Wumpus, and find the gold. 

MAIN FEATURES
1. Adjustable grid size and pit count
2. Manual step and auto play execution modes
3. Live dashboard displaying the agents knowledge base and active percepts
4. Real time inference logging showing the resolution steps
5. Custom retro sci fi terminal interface

HOW TO RUN THE PROJECT
Open the index.html file in any modern web browser. The project runs entirely client side, so there are no servers, dependencies, or build tools required.
Live demo: https://dynamic-wumpus-logic-agent-one.vercel.app/#

PROJECT FILES
index.html contains the main layout and structure
style.css contains all the visual styling and layout rules
script.js contains the game loop, logic engine, and agent decision algorithms
